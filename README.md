# Habit Tracker

A simple Streamlit app to track daily, weekly, monthly, or yearly habits and see your progress over time.

## Features
- Log habits and mark completions with date & time
- View streaks, completion rates, and achievement levels
- Get reminders and habit-stacking suggestions
